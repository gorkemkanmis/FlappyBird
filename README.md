# FlappyBird
 Unity kullanarak geliştirdiğim basit ve eğlenceli bir Flappy Bird klonudur. Oyunda fizik tabanlı uçuş mekaniği, engel üretimi (pipe spawner), skor sistemi, çarpışma kontrolü ve temel UI akışları uygulanmıştır. Bu proje, mobil oyun geliştirme sürecini ve hyper-casual oyun mantığını öğrenmek amacıyla hazırlanmıştır.
